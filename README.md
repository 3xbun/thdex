## ✨ คุณสมบัติหลัก (Features)

- 🃏 **ฐานข้อมูลการ์ดที่ครอบคลุม:** ค้นหาข้อมูลการ์ดโปเกมอนได้อย่างครบถ้วน พร้อมดูสถิติเฉพาะและรูปภาพความละเอียดสูง
- 📁 **จัดการคอลเลกชันของคุณ:** เพิ่ม ลบ หรืออัปเดตการ์ดในครอบครองได้ง่ายๆ
- 🤝 **พื้นที่แลกเปลี่ยน (Community Trades):** เชื่อมต่อกับผู้ใช้ท่านอื่นเพื่อแลกเปลี่ยนการ์ดซ้ำ หรือตามหาการ์ดที่คุณยังไม่มีผ่าน TCG Thailand

## 🚀 การติดตั้ง (Installation)

หากคุณต้องการรันโปรเจกต์นี้บนเครื่องของคุณเอง สามารถทำตามขั้นตอนด้านล่างนี้ได้เลย:

1. **โคลนโปรเจกต์**

   ```bash
   Bash
   git clone [https://github.com/3xbun/thdex.git](https://github.com/3xbun/thdex.git)

   เข้าไปที่โฟลเดอร์ของโปรเจกต์

   ```

2. **เข้าไปที่โฟลเดอร์ของโปรเจกต์**

   ```bash
   Bash
   cd thdex

   ```

3. **ติดตั้ง Dependencies**

   ```bash
   Bash
   npm install

   ```

4. **เริ่มต้นเซิร์ฟเวอร์สำหรับพัฒนา (Development Server)**

   ```bash
   Bash
   npm run dev
   # หรือ yarn dev

   ```

5. **การใช้งาน (Usage)**
   1. เปิดเบราว์เซอร์และเข้าไปที่ http://localhost:5173 (หรือพอร์ตที่ Terminal ระบุไว้)
   2. เริ่มต้นบันทึกคอลเลกชันการ์ด
